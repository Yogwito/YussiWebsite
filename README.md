# YussiWebsite

Sitio web de **Bordados Yussi** — dotación empresarial, uniformes y bordado profesional en Manizales, Colombia. Más de 25 años de experiencia.

## Stack
- [Vite](https://vitejs.dev/) + JavaScript (vanilla)
- [GSAP](https://gsap.com/) para animaciones
- CSS modular por sección

## Desarrollo
```bash
npm install      # instalar dependencias
npm run dev      # servidor de desarrollo
npm run build    # build de producción → dist/
npm run preview  # previsualizar el build
```
